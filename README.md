# Aura Botanics 🌿

A premium, luxury botanical e-commerce storefront designed for a sophisticated shopping experience. Built with performance and aesthetics in mind using modern web technologies.

![Aura Botanics Banner](https://images.unsplash.com/photo-1556228578-0d85b1a4d571?q=80&w=200&auto=format&fit=crop)

## ✨ Features

- **Luxury Product Catalog**: Beautifully rendered product grids and detail pages.
- **Full Shopping Journey**: Integrated cart, wishlist, and secure checkout flow.
- **User Authentication**: Dedicated login and account management pages.
- **Lifestyle Content**: Blog and events modules to engage the community.
- **Responsive Design**: Optimized for all devices, from mobile to ultra-wide displays.
- **High Performance**: Powered by Vite for lightning-fast development and optimized production builds.

## 🛠️ Tech Stack

- **Core**: HTML5, Semantic Markup
- **Logic**: Vanilla JavaScript (ES6+)
- **Styling**: Premium Custom CSS
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/aura-botanics.git
   ```
2. Navigate to the project directory:
   ```bash
   cd aura-botanics
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```
The site will be available at `http://localhost:5173`.

### Production

Build the project for production:
```bash
npm run build
```
The optimized files will be generated in the `dist` folder.

## 📁 Project Structure

- `index.html`: Homepage
- `shop.html`: Product listing
- `product.html`: Individual product details
- `auth.html`: User login/signup
- `cart.html` & `wishlist.html`: Shopping utilities
- `blog.html`: Latest articles and news
- `app.js`: Main application logic
- `index.css`: Global styling and design system
